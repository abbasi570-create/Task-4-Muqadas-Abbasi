# 📷 AI Image OCR App (Python)

This is a simple AI-based Image Text Recognition app built using **Python, Tesseract OCR, and Tkinter GUI**.

---

## 🚀 Features
- Upload image (JPG, PNG)
- Extract text using OCR (Tesseract)
- Display image preview
- Editable text output
- Save extracted text as .txt file
- Simple GUI (no Streamlit needed)

---

Install dependencies:
